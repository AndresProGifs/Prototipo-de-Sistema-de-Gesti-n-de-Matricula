¡ BIENVENIDOS !

Este prototipo hace parte del proyecto MODELO PARA LA IMPLEMENTACIÓN DE UN SISTEMA DE GESTION DE MATRICULA, DESDE UNA INVESTIGACIÓN CIENTÍFICA, BASADO EN LAS METODOLOGÍAS ESTRUCTURADA, ORIENTADA A OBJETOS (UML) Y ÁGIL SCRUM y ha sido desarrollado únicamente con fines académicos, como una muestra sencilla de la manera como se puede diseñar un prototipo de software como elemento importante dentro de la fase de diseño de un sistema de gestion de matricula que se ajuste a las necesidades del la institucion.

En tal sentido, el presente constituye un mero ejemplo de lo que podría ser un prototipo, para lo cual se han usado los requisitos funcionales básicos de un sistema de gestion de matricula, tomando como referente la institucion Eduactiva Isaac Rodriguez Martinez y basado en autores clásicos que se describen en el documento del cual se deriva este prototipo.

Se espera que con este ejemplo y el documento base los interesados puedan sacar a flote su creatividad como para desarrollar nuevos prototipos que se ajusten a las necesidades de las organizaciones en la cual adelanten sus proyectos de desarrollo de software.
